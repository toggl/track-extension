• General
  • Pomodoro Timer
  • Shows a "Start timer" button in the right click menu
  • Start time tracking of latest task when chrome starts
  • Stop time tracking when the browser is closed
  • Opens up edit form right after clicking "Start timer" link
  • Keeps Toggl Button up to date. All updates should arrive in seconds.

  • Shows reminder notification if user has forgot to start timer while working

    Track activity on workdays

  • Shows notification when timer was tracking but user has been idle and allows to discard idle time

  • The timer will be stopped automatically at this time of day

  • Your custom Time Entry formatting, displayed on the Toggl site. Use following tokens:

    Task-ID: $$taskid$$ Descrption: $$description$$

  • Allows user to define Pomodoro Timer interval. Pomodoro timer starts running every time a time entry is started. At the end of the interval the user is shown notification to take a break. User can set an option to stop or continue time tracking when pomodoro timer ends.
